# Connect Four – SODV1202

This project is a console implementation of the classic **Connect Four** game written in **C#**.  
It was created as part of the **SODV1202 course assignment**.

The program allows two players to take turns dropping pieces into a 7×6 board.  
The objective is to connect four pieces in a row (horizontally, vertically, or diagonally) before the other player.

## Features
- Two-player gameplay
- Console board display
- Turn-based player input
- Win detection (horizontal, vertical, diagonal)
- Draw detection when the board is full

## Technologies Used
- C#
- .NET Console Application
